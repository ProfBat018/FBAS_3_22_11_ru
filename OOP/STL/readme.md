# STL - Standart Template Library

### STL состоит из трех частей:
* Контейнеры
* Алгоритмы
* Итераторы

### Контейнеры:
* Последовательные:
    * vector
    * list
    * deque
    * array
    * forward_list
* Ассоциативные:
    * set
    * multiset
    * map
    * multimap
* Адаптеры:
    * stack
    * queue
    * priority_queue

### Алгоритмы - пройдем на следующем занятии

### Итераторы - это указатели на начало и конец контейнера
